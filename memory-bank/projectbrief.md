# Project Brief — ASAK Agency Website

## Identitas Proyek
- **Nama:** ASAK Agency (brand: *asak digital*)
- **Tagline:** "The Anti-Chaos Agency — Done Right. Done On Time."
- **Arti "Asak":** Matang, siap, selesai — merepresentasikan standar kerja agensi
- **Tipe:** Company Profile + CMS Admin Panel

## Tujuan
Membangun website company profile ASAK Agency yang:
1. Menampilkan profil, layanan, portofolio, dan tim secara profesional
2. Menerima pesan/kontak dari calon klien
3. Memiliki panel admin untuk mengelola seluruh konten secara dinamis

## Scope Fitur
### Halaman Publik
- `/` — Home (hero, services preview, portfolio preview, stats, team)
- `/about` — About (story, values, team, stats)
- `/services` — Services (daftar layanan, process steps, why choose us, CTA)
- `/portfolio` — Portfolio (grid dengan filter kategori, stats, featured)
- `/contact` — Contact (form + info kontak + social media)

### Panel Admin (auth required)
- Dashboard dengan statistik ringkas
- CRUD Portfolio (+ trash/restore/force-delete)
- CRUD Team (+ trash/restore/force-delete)
- CRUD Services + Features (+ trash/restore/force-delete)
- CRUD Process Steps
- Manajemen Contact Submissions (baca, tandai, hapus)
- Settings (hero text, about text, stats, contact info, social links)

## Layanan yang Ditampilkan
1. Brand Engineering — Identity, UI/UX, Visual System, Graphic & Video
2. Tech Development — Web, Apps, Custom Software
3. Growth Hacking — Data-Driven Marketing & SEO
4. Photo & Videography — Professional visual content

## Nilai Inti
- Radical Transparency
- Zero-Delay Protocol
- Global Standard
- Definition of Done
